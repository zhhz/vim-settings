vim-settings
============

A quick settings for vim


F2 - toggle NERDTree
F3 - toggle tag list
F5 - toggle paste mode
F6 - toggle column
F7 - toggle background

,bd - delete current buffer
,be - buffer explorer

:sp - horizon split
:vs - vertical split
C-W c - close a window
CTRL + h/j/k/l - move between windows
META + u/i/o/p - resize windows with the mata/option key


,ig - toggle indent
,y  - yangk ring
